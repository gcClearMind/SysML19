3D Inverted Pendulum System Simulator in JAVA

  • invpend.jar: 3D Inverted Pendulum System Simulator (executable JAR)

    • This JAR file demonstrates an inverted pendulum system, designed using MagicDraw with MagicDraw SysML plugin.

    • The default controller gains (k1, k2, k3, k4) can be changed to meet the required response.



    • Disturbance, which is pendulum angle (in Degree), can be used to disturb the system.



    • JRE and J3D are required for this application. J3D can be downloaded from http://java3d.java.net/binary-builds.html.



  • invpend.jar: Execution Instruction (For Microsift Windows XP)

    • 1) Download JRE from http://java.sun.com and install it to your system.

    • 2) Download Java3D from http://java3d.java.net/binary-builds.html and install it to your system.

    • 3) Right-click on invpend.jar file and select "Open With..." and select "Java(TM) Platform SE binary"
